# Softlend Credit Dashboard

## Overview

A React-based dashboard that helps users understand their credit score, identify improvement opportunities, and explore eligible loan offers.

## Features

- Credit Score Dashboard
- Credit Score Gauge
- Potential Score Calculation
- Score Factors Analysis
- Improvement Plan
- Loan Offers Marketplace
- Locked / Unlocked Offers
- Search with Debounce
- Offer Filtering
- Confirmation Modal
- Loading State
- Error State
- Empty State
- Score Simulator
- Responsive Design

## Tech Stack

- React.js
- JavaScript
- CSS Modules
- React Hooks
- JSON Data

## Installation

```bash
npm install
npm start
```

No database migration is required for this project because it uses static JSON data

Run the application:

npm start

## Postman Usage

No API testing required. This project is a frontend React application using JSON data.
